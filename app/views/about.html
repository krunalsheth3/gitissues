GitHub new Issues viewer

This project is a new Issues viewer for GitHub, using Angular.js, Angular material design & angular-bootstrap UI The API documentation is here.

It has a default page showing Issue number & title, Lables, Reporters username & avatar, summary of the description Second page is a Issue details oage which shows Issue titls, state, Labels, Reporters username & avatar & complete summary of the issue.

If there are comments then they are present in a readable format

For all the pages, there is Server side Pagination & sorting available. Wherever search bar is present is a client side search
